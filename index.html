Mermaid Studio

Help Guide

Navigation

  • Drag - Click and drag to pan around the diagram
  • Ctrl + Scroll - Zoom in/out with mouse wheel
  • R key - Reset zoom and position

AI Features

  • Gemini API Key - Enter your key in the header to enable AI
  • Fix with AI - When you see a syntax error, click the purple lightning button to auto-fix
  • Get your free API key at Google AI Studio

Import / Export

  • Import - Load .mmd, .txt, or .md files
  • Export Code - Download the source code as .mmd file
  • SVG - Download vector graphic (scalable, transparent)
  • PNG - Download raster image (with background color)

Display Modes

  • Computer icon - Follows your OS/browser theme automatically
  • Sun icon - Force light mode for the editor
  • Moon icon - Force dark mode for the editor
Tip: The diagram updates automatically as you type (with a small delay). If you see a red error panel at the bottom of the editor, that's where syntax errors appear. You can click "Clear" to dismiss it or "Fix with AI" to auto-correct!
